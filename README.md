# Ataque-de-Brute-Force-de-Senhas-com-Medusa-e-Kali-Linux
Este repositório apresenta a implementação prática de um ambiente controlado de cibersegurança, desenvolvido como parte do desafio proposto pela DIO. O objetivo é explorar e compreender, na prática, ataques de força bruta em diferentes serviços, utilizando ferramentas amplamente presentes no ecossistema de segurança ofensiva.
